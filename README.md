#  King County House Sales Data Prediction
By Sally Musanga Kabole

## Project Overview

This project uses regression modeling to analyze the King County House Sales dataset and provide insights to home owners on factors affecting the sale price of their houses. The aim of this analysis is to build  regression model that can predict house prices with the greatest accuracy possible. 


### Business Problem

Consider a startup in the real estate space that wants to make a successful house sale for their client. In order to price any cliient's home competitively  and get offers from prospective buyers,The startup contacts a data scientist to give insights into what can be done to increase house prices.
### Methods
- Exploring and preprocessing the data
- Building simple and multiple linear regression models in OLS statsmodels, with price as the dependent variable.
